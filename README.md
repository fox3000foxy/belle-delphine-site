# Belle Delphine - Official Website

The official website of Belle Delphine, featuring exclusive content, latest updates, and community access.

## Features

- **Exclusive Videos** - Premium video content and behind-the-scenes footage
- **Photography Sets** - High-quality photoshoots and artistic collections  
- **Official Merchandise** - Unique items and limited edition products
- **Community Access** - Exclusive fan community and direct interactions
- **Premium Content** - Subscription-based content with member perks
- **Personal Connection** - Direct interaction opportunities and personalized content

## Tech Stack

- **React 18** - Modern React with hooks and functional components
- **TypeScript** - Type-safe development
- **Material-UI** - Beautiful React components
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Fast build tool and dev server
- **FontAwesome** - Icon library

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
